Creating Year Slicer 17

1.      Select Slicer from visualization pane
2.      Open ‘Calendar’ table and drag Year in the slicer.
3.      Click on Visual and make these changes: 
4.      Slicer settings, style: Between
5.      Slicer header: Off
6.      Values: Font = Segoe UI Bold, Size = 9, Bold, Color = #666666.
7.      Slider: On, Color = #666666.
8.      Go to General, Properties, 
          Size: Height = 107, 
          Width = 204, 
          Position: Horizontal = 0, 
          Vertical = 80, 
9.      Title: Year, 
           Font = Segoe UI Bold, Size =14, text Color = #666666, 
           background color = #71BFC5, 
           Horizontal alignment = Center. 
           Subtitle = Off, 
           Divider = Off, 
           Spacing = 5 px.
10.    Effects: Background=On, 
           Color = #E6E6E6, 
           Visual border: on, 
           Color = #666666, 
           Shadow = Off
