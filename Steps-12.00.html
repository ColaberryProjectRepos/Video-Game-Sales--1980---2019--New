Creating “.Decades” Calculated Column 12

1-     Go to Data View
2-     Select the “Calendar Table”
3-     Click on Table tools and then Click on New Column
4-     Copy & Paste the DAX Below


DAX Formula:
.Decades = VAR DecadeStart =
    ROUNDDOWN ( 'Calendar'[Date].[Year], -1 )
VAR DecadeEnd = DecadeStart + 9
RETURN
    DecadeStart & " - " & DecadeEnd
