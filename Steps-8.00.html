Import CSV file into Power BI Desktop 8
  1. Download the 'VideoGameSales'  file by clicking the red icon above.
  2. Open Power BI Desktop
  3. Click on Get data button 
  4. Select Text/CSV and click on connect 
  5. Select the 'VideoGameSales.CSV' file and click on Open
  6. A preview of the data will be displayed, click on Load.
