Transforming the data in power Query Editor 9

1.    In this step you will clean and transform the data
2.    Select Transform Data from the Home Tab
3.    Right Click on the Date Column and Select Change Type
4.    Select Text
5.    Select add New Step
6.    Right Click again on the Date Column and Select Change Type
7.    Set the datatype for this column as Date
8.    Select add New Step
9.    Select Close & Apply
10.  Go to Data View from the Left panel
11.  Select the Date Column
12.  Change the Format to “mm/dd/yyyy”
