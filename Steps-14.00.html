Creating all Measures 14


1.      Click on report view
2.      Right click on “VideoGameSales” and select New measure.
3.      In the DAX Formular Copy & Paste each of the below formulas one by one.
4.      Choose the necessary format and decimal for each measure then Save.
 
Measures (Dax Formula):


.YoY% = 
VAR CurrentYearSales = SUM('VideoGameSales'[GlobalSales])
VAR PreviousYearSales = CALCULATE(
    SUM('VideoGameSales'[GlobalSales]), 
    DATEADD('Calendar'[Date], -1, YEAR)
)
RETURN
    IF(ISBLANK(PreviousYearSales), BLANK(), DIVIDE(CurrentYearSales - PreviousYearSales, PreviousYearSales))


For the YoY% Measure, make sure to Format it as “Percentage” instead of “General” as shown in the pic below.
 



.Total Global Sales = SUM(VideoGameSales[GlobalSales])

.Average Global Sales Per Year = AVERAGE(VideoGameSales[GlobalSales])



For BOTH of .Total AND .Average Global Sales Measures, make sure to Format it as “Currency”

as shown in the pic below



.Last Decade DoD =
Var CYear = year(max(Calendar[Date]))
var EODecade = date(roundup(
 if(right(CYear,1)="0",
CYear+1,
CYear)
,-1)-11,12,31)
var SODecade = date(ROUNDDOWN(CYear,-1)-10,1,1) return
CALCULATE (
SUM('VideoGameSales'[GlobalSales]),
all('Calendar'),
DATESBETWEEN (
'Calendar'[Date],
SODecade,
EODecade
)
)


.DoD = 
var CYear = year(max('Calendar'[Date]))
var EODecade = date(roundup(
if(right(CYear,1)="0",
CYear+1,
CYear)
,-1)-1,12,31)
var SODecade = date(ROUNDDOWN(CYear,-1),1,1) return 
CALCULATE (
SUM('VideoGameSales'[GlobalSales]),
DATESBETWEEN (
'Calendar'[Date],
SODecade,
EODecade
)
) 


.DoD% = DIVIDE([.DoD]-[.Last Decade DoD],[.Last Decade DoD], 0)


Just like what we did with the YoY% Measure, let’s make we format the DoD% as “Percentage” instead of “General” as shown in the pic below.

 



.ArrowForDOD% = SWITCH(TRUE(), [.DoD%] <0, UNICHAR(9662),[.DoD%] =0, UNICHAR(9666), UNICHAR(9652))

.ArrowForYOY% = SWITCH(TRUE(), [.YoY%] <0, UNICHAR(9662),[.YoY%] =0, UNICHAR(9666), UNICHAR(9652))

.ColorForArrowDoD = IF([.ArrowForDOD%] = UNICHAR(9652),"Green",IF([.ArrowForDOD%] = UNICHAR(9662),"Red","Yellow"))

.ColorForArrowYoY = IF([.ArrowForYOY%] = UNICHAR(9652),"Green",IF([.ArrowForYOY%] = UNICHAR(9662),"Red","Yellow"))
