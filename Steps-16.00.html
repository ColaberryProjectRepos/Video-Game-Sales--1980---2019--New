Adding Project Title and Logos. 16

1.      In the report view, Select Inset ribbon > Text box.
         -Type Project Name “Video Game Sales (1980 – 2019)”
         -Highlight the Text and adjust the Size to 28, Font Segoe (Bold) Center, Color=#666666
         -In the Formatting Tab, click on General, Properties, Size: Height=74, Width=994, Position: Horizontal = 204, Vertical = 5.


2.      Click on Insert ribbon then Image and select the Colaberry logo.
         - In the Formatting Tab, click General, Properties, Size: Height = 80, Width = 204, Position: Horizontal = 0, Vertical = 0, Background: Off. 


3.      Click on Insert ribbon then Image and select the VGS logo.
         - In the Formatting Tab, click General, Properties, Size: Height = 80, Width = 80,  Position:  Horizontal = 1198, Vertical = 0, Background: Off.
