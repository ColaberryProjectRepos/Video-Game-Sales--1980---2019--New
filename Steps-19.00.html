Adding Indicators to Sales YoY and DoD KPIs 19


1.      To create the YoY indicator, click on an empty space on the canvas, and from the visualizations pane select Card.
2.      Drag .ArrowForYOY% to the fields value box.       
3.      Go to format visual
4.      Click on Visual and make these changes:
5.      Callout Values: Font = DIN, 
          Size = 45, 
          color: Set the color by clicking on fx (conditional formatting).
          format style= Field Value,
          What field should we base this on? = .ColorForArrowYoY,
          Display units = Auto, 
          Value decimal = Auto
6.      Category label: off.
7.      Go to General, Properties, Size: 
          Height = 63, 
          Width = 69, 
          Position: 
          Horizontal = 149, 
          Vertical = 456. 
8.      Title: Off,
9.      Effects: Background = Off
10.   Tooltips = Off
